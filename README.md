# GPS-Tracker

**🛠️ Komponen yang Dibutuhkan**
1.  Mikrokontroler ESP32 (atau sejenisnya)
2.  Layar OLED SSD1306 (I2C)
3.  3x LED (Merah, Kuning, Hijau) beserta Resistor
4.  1x Buzzer (Active Buzzer)
5.  Kabel Jumper secukupnya
6.  Input sinyal "Jalan" dan "Berhenti" (Bisa berupa sensor, tombol/push button, atau relay dari GPS Tracker)


**📚 Library yang Diperlukan**
Sebelum mengunggah (upload) kode ini melalui Arduino IDE, pastikan Anda sudah menginstal *library* berikut melalui **Library Manager**:
1.  `Adafruit GFX Library` oleh Adafruit
2.  `Adafruit SSD1306` oleh Adafruit
